Кастомные иконки меню вложений

Папка: assets/icons/custom

Поддерживаемые имена (любой из форматов: .svg, .png, .webp, .jpg, .jpeg):
- attach_photo
- attach_video
- attach_poll
- attach_file

Примеры:
- assets/icons/custom/attach_photo.svg
- assets/icons/custom/attach_video.png

Если файла нет, используется стандартная иконка из assets/icons.
