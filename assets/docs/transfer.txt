СОГЛАСИЕ НА ПЕРЕДАЧУ ДАННЫХ В FIREBASE
МЕССЕНДЖЕР TEN DEM

Версия от 1 марта 2026 года

1. ОБЩИЕ ПОЛОЖЕНИЯ

1.1. Настоящим я подтверждаю своё согласие на передачу и хранение моих персональных данных в облачной инфраструктуре Firebase.

1.2. Firebase — это платформа облачных вычислений, разработанная компанией Google LLC (США).

1.3. Передача данных необходима для функционирования мессенджера Ten Dem.


2. КАКИЕ ДАННЫЕ ПЕРЕДАЮТСЯ

2.1. В Firebase передаются следующие данные:
    • Номер телефона (для аутентификации)
    • Профиль пользователя (имя, username, аватар)
    • Сообщения и медиафайлы
    • Метаданные переписки (время, статус доставки)
    • Токены устройств для push-уведомлений
    • Техническая информация (устройство, ОС, версия приложения)


3. ЦЕЛЬ ПЕРЕДАЧИ

3.1. Данные передаются для следующих целей:
    • Хранение и синхронизация сообщений между устройствами
    • Аутентификация пользователей
    • Отправка push-уведомлений
    • Резервное копирование данных
    • Аналитика использования Сервиса
    • Предотвращение злоупотреблений


4. МЕСТО ХРАНЕНИЯ

4.1. Данные хранятся на серверах Firebase/Google.

4.2. Сервера могут располагаться в различных регионах мира, включая:
    • Европейский Союз
    • Соединённые Штаты Америки
    • Азиатско-Тихоокеанский регион

4.3. Конкретный регион хранения определяется автоматически для оптимальной производительности.


5. ЗАЩИТА ДАННЫХ

5.1. Firebase применяет следующие меры защиты:
    • Шифрование данных при передаче (TLS 1.3)
    • Шифрование данных при хранении (AES-256)
    • Регулярный аудит безопасности
    • Сертификация ISO 27001, SOC 2, GDPR

5.2. Google обязуется соблюдать конфиденциальность и не использовать данные в коммерческих целях без согласия.


6. ПРАВОВЫЕ ОСНОВАНИЯ

6.1. Передача данных осуществляется на основании:
    • Настоящего согласия Пользователя
    • Договора с Google Cloud Platform
    • Политики конфиденциальности Firebase

6.2. Google LLC обязуется соблюдать применимое законодательство о защите данных.


7. РИСКИ

7.1. Я понимаю следующие риски:
    • Данные могут быть доступны государственным органам США по запросу
    • Изменения в политике Firebase могут повлиять на обработку данных
    • Прекращение сервиса Firebase может повлиять на работу Ten Dem

7.2. Разработчики Ten Dem предпринимают меры для минимизации рисков.


8. ПРАВА ПОЛЬЗОВАТЕЛЯ

8.1. Я имею право:
    • Получить информацию о передаваемых данных
    • Отозвать настоящее согласие (путём удаления аккаунта)
    • Экспортировать свои данные
    • Подать жалобу в уполномоченный орган


9. ОТЗЫВ СОГЛАСИЯ

9.1. Согласие может быть отозвано в любое время путём удаления аккаунта.

9.2. После отзыва данные будут удалены в срок до 90 дней.

9.3. Отзыв согласия прекращает возможность использования Сервиса.
