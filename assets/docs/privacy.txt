1. ОБЩИЕ ПОЛОЖЕНИЯ

1.1. Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей мессенджера Ten Dem.

1.2. Политика разработана в соответствии с Федеральным законом № 152-ФЗ «О персональных данных» от 27.07.2006.

1.3. Используя Сервис, Пользователь подтверждает, что ознакомлен с настоящей Политикой и согласен на обработку своих персональных данных.


2. КАКИЕ ДАННЫЕ МЫ СОБИРАЕМ

2.1. Данные, предоставляемые Пользователем:
    • Номер телефона (обязательно)
    • Имя и фамилия (по желанию)
    • Username (по желанию)
    • Аватар (по желанию)
    • Статус «О себе» (по желанию)

2.2. Данные, собираемые автоматически:
    • Информация об устройстве (модель, ОС, версия приложения)
    • IP-адрес и данные о подключении
    • Время и дата активности
    • Данные о контактах (при предоставлении доступа)

2.3. Содержимое переписки:
    • Текстовые сообщения
    • Медиафайлы (фото, видео, аудио)
    • Голосовые сообщения
    • Пересылаемые файлы


3. ЦЕЛИ СБОРА ДАННЫХ

3.1. Мы используем данные для следующих целей:
    • Идентификация Пользователя в Сервисе
    • Обеспечение обмена сообщениями между пользователями
    • Улучшение работы и функциональности Сервиса
    • Предотвращение мошенничества и злоупотреблений
    • Техническая поддержка пользователей
    • Отправка уведомлений о работе Сервиса


4. ХРАНЕНИЕ ДАННЫХ

4.1. Персональные данные хранятся на защищённых серверах до момента удаления аккаунта Пользователем.

4.2. После удаления аккаунта данные хранятся до 90 дней, после чего подлежат безвозвратному удалению.

4.3. Резервные копии данных могут храниться до 180 дней для обеспечения восстановления Сервиса.

4.4. Мы используем облачную инфраструктуру Firebase (Google LLC) для хранения данных.


5. ЗАЩИТА ДАННЫХ

5.1. В Сервисе применяются следующие меры защиты данных:
    • Шифрование данных при передаче по протоколу TLS/SSL (HTTPS)
    • Шифрование данных при хранении на серверах
    • Регулярное резервное копирование инфраструктуры
    • Контроль доступа через систему аутентификации
    • Автоматический мониторинг безопасности

5.2. Защита данных обеспечивается с использованием облачной инфраструктуры Firebase (Google LLC), которая соответствует международным стандартам безопасности (ISO 27001, SOC 2).

5.3. Доступ к персональным данным имеют только уполномоченные лица (разработчики Сервиса) в целях технической поддержки.

5.4. Мы регулярно обновляем приложение для устранения уязвимостей и повышения безопасности.
    • Мониторинг безопасности
